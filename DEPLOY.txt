어떤 컴퓨터에서든 접속하게 만드는 방법

핵심:
- 이 프로그램은 이제 클라우드 서버에 올릴 수 있는 형태입니다.
- 배포가 끝나면 https://주소 하나로 어디서든 접속하고 로그인할 수 있습니다.
- 로컬 Wi-Fi, 방화벽, IP 주소를 신경 쓰지 않아도 됩니다.

배포할 때 필요한 설정:
1. Start command
   npm start

2. Environment variables
   MEMBERSHIP_ADMIN_USER=원하는_관리자_아이디
   MEMBERSHIP_ADMIN_PASSWORD=원하는_로그인_비밀번호
   MEMBERSHIP_DATA_SECRET=아주_긴_암호화_비밀키
   MEMBERSHIP_DATA_DIR=지속저장소_경로

주의:
- MEMBERSHIP_ADMIN_USER는 관리자 아이디입니다.
- MEMBERSHIP_ADMIN_PASSWORD는 로그인 비밀번호입니다.
- MEMBERSHIP_DATA_SECRET은 데이터 암호화 키입니다.
- MEMBERSHIP_DATA_DIR은 선택사항이지만, Render Disk 같은 지속 저장소를 연결하면 그 경로를 넣으세요.
- MEMBERSHIP_DATA_SECRET을 잃어버리거나 바꾸면 기존 암호화 데이터를 읽을 수 없습니다.
- 공개 운영에서는 반드시 2468 대신 긴 비밀번호를 쓰세요.

데이터 저장:
- 회원 데이터는 data/membership-data.enc.json에 암호화되어 저장됩니다.
- 클라우드 서버에서는 이 data 폴더가 계속 보존되는 저장소로 연결되어야 합니다.
- 저장소가 보존되지 않는 무료 서버에 올리면 서버가 재시작될 때 데이터가 사라질 수 있습니다.

가장 쉬운 진행:
1. 이 membership-manager 폴더를 클라우드 서버에 업로드합니다.
2. 환경 변수 2개를 설정합니다.
3. 실행 명령을 npm start로 설정합니다.
4. 배포된 https 주소로 접속합니다.
